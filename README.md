# Reactの導入手順

React ver 15.6.1  

## 公式

https://facebook.github.io/react/  

## Reactのインストール

```sh
npm init
npm install --save react react-dom
```

## BABEL

```sh
npm install --save-dev babel-cli babel-preset-react babel-preset-es2015
echo '<h1>Hello, world!</h1>' > index.js
```

.babelrcを作成する。
